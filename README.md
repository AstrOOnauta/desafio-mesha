## 💻 Linguagens e tecnologias utilizadas

    - ReactJS
    - Typescript
    - Styles-Components
    - React-icons lib
    - React-Modal lib
    - Hooks
    - localStorage
    - Open Weather API
    - Spotify API
<hr>

## Instalação

    - Clonar repositorio
    $ git clone https://github.com/AstrOOnauta/desafio-mesha.git

    - Instalar dependências
    $ npm install

    - Rodar aplicativo
    $ npm start
